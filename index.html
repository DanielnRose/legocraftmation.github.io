Step 1. Install Virtual Box

Download VirtualBox for your operating system. For this tutorial we will focus on Windows. If you are on MacOS download OS X hosts. After downloading open the VirtualBox installer to finish installing.

Step 2. Download Ubuntu Desktop

Download Ubuntu Desktop and make sure you know the location of this file. For this tutorial we will use Ubuntu, but you may use other Linux distro of your choice.

Step 3. Create Ubuntu VM

Open VirtualBox and click on New to create a new VM (Virtual Machine).

Step 3.1. Name and Operating System

Give your VM a name. Make sure type is Linux and version is Ubuntu (64-bit) or the Linux distro of your choice.

Step 3.2. Memory Size

VM requires memory. Give at least 2GB (2048 MB) and hit next.

Step 3.3. Virtual Hard Disk

Select create a virtual hard disk now and hit create. Next select VDI (VirtualBox Disk Image). Then select Dynamically allocated.

Step 3.4. File Location and Size

You can change your file location if you want. Give at least 15 GB for the size. Then hit create.

Step 4. Add Ubuntu Image to VM

From the previous step an Ubuntu VM should be created with the name you provided in step 3.1. Select it and click the settings button.

Step 5. Image Location

1. Click on Storage.
2. Click on Adds optical drive.
3. Click Add
4. Window will appear. Go to the location of the file from step 2, select it and hit open.
5. Click Choose then hit OK.

Step 6. Start VM

Make sure your VM is selected and hit the Start button to start the VM. A new window should appear.

Step 7. Select Start-up Disk

Make sure to select the virtual optical disk file you added in step 5. If the image is not there click on the folder and select the disk file. Then Click Start.

Step 8. Install Ubuntu

Select your language and click Install Ubuntu. Next select your keyboard layout and hit continue.

Step 9. Updates and Software

If you want to save storage then select Minimal installation. Otherwise keep everything default and click continue.

Step 10. Installation Type

Erase disk and install Ubuntu should be selected. If not select it and hit Install. A window should pop-up. Ignore the warning and click continue. Then select your timezone and hit continue.

Step 11. Create User Account

Give your computer a name and password. Then click continue. The installation process will take 5-10 minutes depending on your computer. After installation is done, a window will pop-up. Click on Restart Now.

Step 12. Remove Installation Medium

Ignore this message and press ENTER on your keyboard.

Step 13. Login

After Ubuntu restarts. You should be in the Login screen. Click on your account and enter the password you created in step 11. Then press ENTER.

Step 14. Logged in Succesfully

If you entered your password correctly then you should be logged in. You can connect your accounts if you want otherwise hit skip and keep hitting next unless you want to change the default settings.

Step 15. Unbutu Desktop Ready

You should now be in the desktop. You have successfully created your own Linux Ubuntu Virtual Machine (a computer within a computer).
