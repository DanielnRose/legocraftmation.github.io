Step 1. Install Virtual Box

Download VirtualBox for your operating system. For this tutorial we will focus on Windows. If you are on MacOS download OS X hosts. After downloading open the VirtualBox installer to finish installing.

Step 2. Download Ubuntu Desktop

Download Ubuntu Desktop and make sure you know the location of this file.

Step 3. Create Ubuntu VM

Open VirtualBox and click on New to create a new VM.

Step 3.2 Name and Operating System

Give your VM a name. Make sure type is Linux and version is Ubuntu (64-bit).

Step 3.3 Memory Size

VM requires memory. Give at least 2GB (2048 MB).

Step 3.4 Virtual Hard Disk

Select create a virtual hard disk now and hit create. Next select VDI (VirtualBox Disk Image). Then select Dynamically allocated.

Step 3.5 File Location and Size

You can change your file location if you want. Give at least 15 GB for the size. Then hit create.

Step 4. Add Ubuntu Image to VM

From the previous step a Ubuntu VM should be created with the name you provided in step 3.2. Select it and click the settings button.

Step 4.2 Image Location

1. Click on Storage
2. Click on Adds optical drive
3. Click Add
4. Window will appear. Go to the location of the file from step 2 and click open.
5. Click Choose

Step 5. Start VM

Make sure your VM is selected and hit the Start button to start the VM. A new window should appear.

Step 6. Select Start-up Disk

Make sure to select the virtual optical disk file you added in step 4.2 If the image is not there click on the folder and select the disk file.

Step 7. Install Ubuntu

Select your lanauge and click Install Ubuntu. Next select your keyboard layout hit continue.

Step 8. Updates and Software

Keep default. If you want to save storage then select Minimal Installation. Click continue.

Step 9. Installation Type

Erase disk and install Ubuntu should be selected. If not select it and hit Install. A window should popup ignore the warning and click continue. Then select your timezone and hit continue.

Step 10. Create User Account

Give your computer a name and password. Then click continue. The installation process will take 5-10 minutes depending on your computer. After installation is done a window will popup. Click on Restart Now.

Step 11. Remove Installation Medium

Ignore this message and press ENTER on your keyboard.

Step 12. Login

After Ubuntu restarts. You should be in the Login screen and see your user account. Click on it and enter the password you created in step 10. Then press ENTER.

Step 13. Logged in Succesfully

If you entered your password correctly then you should be logged in. You can connect your accounts if you want otherwise hit skip and keep hitting next unless you want to change default settings.

Step 14. Unbutu Desktop Ready

You have created your own Linux Ubuntu Virtual Machine (a computer within a computer).
